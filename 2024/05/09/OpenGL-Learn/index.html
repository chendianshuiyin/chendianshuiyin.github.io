OpenGL-Learn

学习opengl的一些记录和流程
练习c++

opengl a start

屏幕分辨率-空间离散化

OpenGL会找点最近的像素进行绘制

显示器
显示器每个像素又由三个不同颜色led组成
每个颜色可以用256个数字调节

屏幕的Z方向
OpenGL规定朝向屏幕里面为z轴正方向
z轴服务于OpenGL的深度测试功能，点的z越大也就说明它陷入屏幕越深

深度测试
OpenGL模拟现实中的遮挡效果，给像素增加深度值信息
被前景遮挡的像素就不用绘制，这种图像称为RGBD图
现有的PNG格式支持alpha透明度通道，深度值D通常会保存到alpha通道里

clip
超出x,y轴范围的地方需要裁剪

由于openGL的深度缓冲是有限精度，因此z轴方向也会裁剪

世界空间坐标系
三位图像

指定一个点的坐标
用glVertex3f函数，OpenGL默认会将世界坐标直接当屏幕空间坐标

用glBegin(GL_xxx)来指定要给绘图模式，用glVertex3f指定该模式下点的坐标，用glEnd()结束绘制

状态机模型

OpenGL是状态机模型，一次设置永久生效，直到再次设置或者取消

使用CHECK_GL宏来检测绘制过程是否出错
